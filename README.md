Snake Water Gun GUI Game 🐍💧🔫
A simple Python GUI Snake-Water-Gun game built using Tkinter.
This project was created as a learning project while practicing Python programming.
---
Features

• Graphical User Interface using Tkinter
• Snake Water Gun game logic
• Scoreboard system
• Game history tracking
• Play again button
• Simple AI opponent logic

Technologies Used

• Python
• Tkinter (Python GUI Library)
---
How to Run the Game

1. Clone this repository

git clone https://github.com/vikramcodex0/snake-water-gun-gui.git

2. Open the project folder
3. Run the game

python snake_water_gun_game.py
